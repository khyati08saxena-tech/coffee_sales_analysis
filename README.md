# Sales Data Analysis using Python

## Project Overview
This project analyzes sales data using Python and Jupyter Notebook to understand
payment type distribution and sales patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Payment type count
- Bar chart visualization
- Pie chart visualization

## Outcome
Identified most used payment method and visualized data distribution.
